# Markdown AI Cross-Review Skill

A Markdown-based AI Cross-Review skill for comparing analysis from multiple AI agents, identifying agreements and disagreements, evaluating risks, and producing a practical final verdict.

This workflow is designed for use cases such as code review, architecture review, performance analysis, security review, requirement analysis, and technical decision-making.

## What it does

- Reads Markdown analysis files from multiple AI agents
- Extracts key claims, recommendations, assumptions, and risks
- Compares agreement and disagreement between agents
- Evaluates evidence quality and correctness risk
- Produces a prioritized action plan
- Generates a final verdict for decision-making
- Supports report mode, dialogue mode, and visual summary mode
